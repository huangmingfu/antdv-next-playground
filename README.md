# Antdv Next Playground 🎮

在线实时编辑和预览 Antdv Next 组件，无需安装任何依赖！

[![Antdv Next](https://www.antdv-next.com/assets/antdv-next-Cum7m2ZU.svg)](https://www.antdv-next.com)
[![Vue.js](https://img.shields.io/badge/Vue.js-3.x-brightgreen)](https://vuejs.org/)
[![License](https://img.shields.io/github/license/huangmingfu/antdv-next-playground)](LICENSE)

## 🚀 演示地址

[Antdv Next Playground](https://huangmingfu.github.io/antdv-next-playground)

## 🌟 特性

- 💡 **即时预览**: 实时编辑代码并查看结果
- 🧩 **多版本支持**: 支持切换 Vue、Antdv Next 和 TypeScript 版本
- 📦 **零配置**: 无需安装依赖，开箱即用
- 🔧 **在线调试**: 快速测试组件和代码片段
- 🌐 **在线分享**: 生成可共享的 URL 链接
- 🎨 **现代化界面**: 基于 Antdv Next 构建的美观界面

## 🛠️ 技术栈

- [Vue 3](https://github.com/vuejs/core) - 渐进式 JavaScript 框架
- [Antdv Next](https://github.com/antdv-next/antdv-next) - Vue 3.5 UI 库
- [@vue/repl](https://github.com/vuejs/repl) - Vue 官方 REPL 组件
- [UnoCSS](https://github.com/unocss/unocss) - 即时按需原子 CSS 引擎
- [VueUse](https://github.com/vueuse/vueuse) - Vue Composition API 实用工具集

## 🎯 使用场景

- 快速测试 Antdv Next 组件
- 在线调试 Vue 代码
- 分享可复现的代码示例
- 学习 Antdv Next 的交互式教程
- 创建在线演示和原型

## 🚀 快速开始

### 在线使用

直接访问 [Antdv Next Playground](https://huangmingfu.github.io/antdv-next-playground) 即可开始使用。

### 本地开发

```bash
# 克隆项目
git clone https://github.com/huangmingfu/antdv-next-playground

# 进入项目目录
cd antdv-next-playground

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```
